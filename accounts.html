Accounts
One-time Google Cloud setup. Your data is stored only in your own Google Drive — no third-party servers.
Step 1 — Google Cloud Console

Go to console.cloud.google.com and create a new project (or use an existing one).

In APIs & Services → Library, search for and enable the Google Drive API.

In APIs & Services → OAuth consent screen, set type to External, fill in app name (e.g. "My Accounts"), and add your own Google account as a test user.

Step 2 — Create OAuth Client ID

Go to APIs & Services → Credentials → Create Credentials → OAuth Client ID.

Choose Web application. Under Authorised JavaScript origins, add the exact URL where this file lives. If opening locally add http://localhost. If hosted, add your full URL.

Leave redirect URIs blank. Click Create and copy the Client ID.

Step 3 — Paste Client ID here

Saved only in your browser's localStorage. Never transmitted anywhere except Google's own servers during OAuth.
